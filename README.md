# Chrome Extension Boilerplate

Starter template for a Chrome extension with a default popup

## Tech stack

- [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite 8](https://vitejs.dev/)
- [Tailwind CSS 4](https://tailwindcss.com/)

## Getting Started

```bash
# Install dependencies
pnpm install

# Run the development server
pnpm run dev

# Build for production
pnpm run build
```

## Usage

1. Clone or download this repository
2. Run `pnpm install`
3. Add PNG icons to `public`: `icon16.png`, `icon32.png`, `icon48.png`, `icon128.png`
4. Run `pnpm run build`
5. In Chrome, open `chrome://extensions/`, click **Load unpacked**, and select the **dist** folder
